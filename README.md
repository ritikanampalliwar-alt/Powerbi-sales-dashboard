# Powerbi-sales-dashboard
Interactive Power BI dashboard built using Sample Superstore data to analyze sales performance, profit trends, quantity sold, and category-wise insights with KPIs and visuals for business decision making.

#  Key KPIs
- **Total Sales**
- **Total Profit**
- **Total Quantity Sold**
- **Average Order Value (AOV)**

##  Key Insights
- Furniture category shows **negative profit**, indicating high cost or discount impact.
- Technology category contributes higher **average order value**.
- Texas records the **highest quantity sold** among states.
- Monthly sales trend highlights **seasonal demand patterns**.
